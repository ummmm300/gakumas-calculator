学マスサポカ計算機

学マスのサポートカード編成を自動で最適化するツールです。

学マス計算機 URL
https://gakumas-calculator-um.streamlit.app/

 # 使い方
1. 上記URLを開き、ボタンを押して「owned_cards_template.csv」をダウンロード
2. Google Spreadsheetを開いて、
　「ファイル」→「インポート」→「アップロード」と進んで、先ほどダウンロードしたcsvを開く
3. "owned"列と"limit_break"列に適切な数字を入れる
   (Google Spreadsheet内で詳しく説明しています)
4. サポカの所持状況を編集し終えたら、csvファイルとしてダウンロードする
5. 「学マス計算機」を開き、プラン、傾向、SP枚数の上限、下限を設定し、先ほど編集したcsvファイルをアップロードする
6. 計算ボタンを押す

 # 機能
- レンタル1枠込みで最適化
- SP枚数の指定が可能
- 編成パターン比較（3/3/0・3/2/1・2/2/2）
